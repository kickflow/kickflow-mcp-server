分かりやすいコミットメッセージとともに、Gitリポジトリに変更をコミットします。

## 手順

1. 変更内容を確認します。
    - ユーザーから依頼された作業の最後にコミットする場合は、その作業内容を確認してください。
    - それ以外の場合は、`git status`を実行して、変更されたファイルを確認します。
2. 変更内容をステージングします。
    - すべての変更をステージングするには、`git add .`を実行します。
    - 特定のファイルのみをステージングする場合は、`git add <ファイル名>`を実行します。
3. コミットメッセージを作成します。
    - コミットメッセージは、変更内容を簡潔に説明するものにしてください。
    - コミットメッセージは英語で書いてください。
